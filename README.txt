Projekt na studia z przedmiotu Przetwarzanie danych masowych chmurze. 

Aplikacja w pythonie z wykorzystaniem FastApi i biblioteki sqlalchemy, obsługiwana przez Azure App Service i łącząca się z bazą danych Azure SQL Database. Dane z bazy danych są wysyłane do usługi Power BI, gdzie tworzone są raporty oraz wykresy.
